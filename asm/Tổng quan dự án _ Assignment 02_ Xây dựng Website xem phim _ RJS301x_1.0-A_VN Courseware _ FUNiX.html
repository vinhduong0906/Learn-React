Tổng quan dự án

Ở bài Assignment 02 này, bạn sẽ áp dụng các kiến thức đã học để xây dựng giao diện cho một Website xem phim. Trang Web này sẽ bao gồm 2 chức năng tìm xem thông tin cũng như tìm kiếm phim theo từ khóa. Để có được dữ liệu về các bộ phim, bạn sẽ sử dụng 1 API có tên là The Movie Database API. Ta sẽ có hai giao diện như sau:

1. Trang xem thông tin phim (Browse Page)

2. Trang tìm kiếm (Search Page)

Bạn sẽ cần tải Starter Code tại link sau. File Start Code này sẽ có cấu trúc như sau:

Assignment 02 (Starter Code)
├─ package.json
├─ public
└─ src
   ├─ App.js
   ├─ index.css
   ├─ index.js
   ├─ pages
   │  ├─ browse
   │  │  └─ Browse.jsx
   │  └─ search
   │     └─ Search.jsx
   └─ serviceWorker.js

Bạn cần chỉnh sửa các đoạn code ở các file:

  • Browse.jsx: File code giao diện cho trang xem thông tin
  • Search.jsx: File code giao diện cho trang tìm kiếm phim

Lưu ý: Bạn có thể chỉnh sửa file Starter Code hay bổ sung thêm các file để hoàn thành các yêu cầu cho bài Assignment này.
